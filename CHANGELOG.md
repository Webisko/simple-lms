# Changelog

## [1.0.0]

First public release.

### Added

- Course structure: Courses → Modules → Lessons
- Access control (including WooCommerce-based purchasing flow when WooCommerce is active)
- Builder support: Elementor widgets and Bricks elements for course/lesson pages
- Progress tracking (lesson completion)
- REST API under `simple-lms/v1`
- Multilingual compatibility and translatable UI strings
- Privacy tools integration and uninstall options
