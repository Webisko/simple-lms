Simple LMS is a WordPress plugin for selling and delivering online courses.

== Description ==

Simple LMS provides a clean course structure (courses → modules → lessons), access control (typically via WooCommerce purchases), and builder-friendly integrations for Elementor and Bricks.

Highlights:

* Course content hierarchy: Course → Module → Lesson
* WooCommerce integration for selling courses (optional)
* REST API endpoints under the `simple-lms/v1` namespace
* Builder-friendly integrations for Elementor and Bricks

== Installation ==

1. Upload the `simple-lms` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. (Optional) Install and activate WooCommerce if you want to sell courses.

== Frequently Asked Questions ==

= Do I need WooCommerce? =

No. WooCommerce is only required if you want to sell courses and tie access to purchases.

= Does it work with Elementor or Bricks? =

Yes. Elementor and Bricks integrations are optional.
